# 📱 WoeidChat Dashboard - Complete User Guide

## 🎯 Overview

The WoeidChat Dashboard is your command center for global messaging. It's designed with the same clean, intuitive interface as WhatsApp and Telegram.

---

## 🌟 Main Features

### **1. Messages Tab** (Default)
Your conversation history with contacts.

#### What You Can Do:
- ✅ View all active chats
- ✅ See unread message count
- ✅ Quick access to recent conversations
- ✅ Online/offline status indicator
- ✅ Last message preview
- ✅ Sort by most recent

#### How to Use:
1. Open WoeidChat
2. "Messages" tab is selected by default
3. Click any chat to open conversation
4. Type message and press Enter (or click send)

---

### **2. Discover Tab** (Global User Search)
Find and connect with users around the world.

#### What You Can Do:
- ✅ Browse suggested users
- ✅ View user profiles
- ✅ Add contacts with one click
- ✅ See online status
- ✅ Discover new people

#### How to Use:
1. Click "Discover" tab
2. Browse user cards
3. Click "Add" to add anyone to contacts
4. Start chatting!

#### User Card Shows:
- 👤 Avatar (emoji icon)
- 👤 Full name
- 👤 @username
- 👤 "Add" button
- 👤 Online indicator

---

### **3. Contacts Tab** (Your Friends)
Manage all your contacts in one place.

#### What You Can Do:
- ✅ View all contacts
- ✅ See contact status (online/offline)
- ✅ Quick chat access
- ✅ Sort by name/status
- ✅ Search within contacts

#### How to Use:
1. Click "Contacts" tab
2. See all your added contacts
3. Click any contact to start chatting
4. Add new contacts from Discover tab

## 🆘 Troubleshooting

### **Messages Not Sending**
1. Check internet connection
2. Verify token is valid
3. Refresh page
4. Clear browser cache
5. Try different browser

### **Notifications Not Working**
1. Check browser permissions
2. Enable in settings
3. Check "Do Not Disturb"
4. Try reloading page

### **Can't Find User**
1. Verify username spelling
2. User might be new
3. User might have left
4. Try full search

### **Call Won't Connect**
1. Both users must be online
2. Check internet speed
3. Try refreshing
4. Try different browser
5. Contact support
