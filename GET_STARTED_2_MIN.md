# 🚀 WoeidChat - Start Using in 2 Minutes

## ⚡ Super Quick Start

### **Step 1: Open App** (30 seconds)
```
🌐 Go to: http://localhost:8765/static/login.html
```

### **Step 2: Create Free Account** (30 seconds)
```
1. Click "Register"
2. Fill in:
   • Name: Your name
   • Email: your@email.com
   • Username: @yourname
   • Password: AnyPassword123
   • Hint: Your password hint
3. Click "Create Account"
```

### **Step 3: Login** (30 seconds)
```
1. Enter your username
2. Enter your password
3. Click "Sign in"
4. Dashboard loads! ✅
```

## 💬 Send Your First Message

### **To Existing Contact:**
```
1. Click on chat in left sidebar
2. Type message in input box
3. Press Enter (or click 📤)
4. Message sent! ✅
```

### **To New User:**
```
1. Click "Discover" tab
2. Click "Add" on any user
3. User appears in Messages
4. Click to chat
5. Send message! ✅
```

## 🌍 Find People to Chat With

### **Browse Suggested Users** (Discover Tab)
```
1. Click "Discover" tab
2. See user cards with avatars
3. Click "Add" to add anyone
4. They appear in your chats
```

### **Search Specific User** (Search Box)
```
1. Type username in search box
2. Real-time results appear
3. Click result
4. Start chatting!
```

### **View Contacts** (Contacts Tab)
```
1. Click "Contacts" tab
2. See all your friends
3. Click to chat
```
